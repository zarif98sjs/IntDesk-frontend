# **`IntDesk`**

Your go-to interview preparation platform

# Setup and Start

- `yarn install`
- `yarn start`

# Following the template

- include `<Navbar />` in all your apps, to use the same navbar. For example, look at the `App.js` file

  ```js
  export default function App() {
    return (
      <div className="">
        <Navbar />
      </div>
    );
  }
  ```
